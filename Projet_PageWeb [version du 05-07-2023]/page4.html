Difficultés rencontrées

  • Partage d'un seul fichier CSS : il peut y avoir des problèmes de conflits lorsque deux ou plusieurs déclarations sont contradictoires. Par exemple, une déclaration demande que la couleur du texte soit verte et une autre déclaration demande que la couleur du texte soit bleue ;
  • Difficultés pour relire le code python de quelqu'un autre.

Les avantages du travail de groupe

  • le partage du travail en fonction des compétences de chacun(e)s (surtout lorque le temps est limité) ;
  • l'entraide permet de gagner en rapidité et efficacité ;
  • le partage d'idées permet de gagner en créativité.

Pistes d'amélioration

  • créer une interface graphique avec Tkinter ;
  • créer une version mobile de la page ;
  • faire jouer l'ordinateur.
